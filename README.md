# GOtanzhen
使用GO语言写的服务器探针

# 文件说明

app源代码 为uni-app代码导入直接打包即可

探针源代码 为go写的客户端与服务端代码

编译后客户端 是编译后的linux64客户端(直接使用即可,可能需要给权限)
